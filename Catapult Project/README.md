# FutureCash Health

FutureCash Health is a prototype product concept that combines:

- Finance forecasting for healthcare and biotech operators
- Clinical trial matching for patient and site coordination

## Prototype

This repository currently includes a zero-dependency front-end demo with:

- A cash runway and revenue projection panel
- Portfolio metrics for trial programs
- An interactive patient intake and patient-to-trial matching experience
- Search filters, nearby-trial logic, and visible risk disclaimers
- A starter medical-note extraction flow for structured fields
- Alert messaging for recruiting status and nearby or remote matches
- Mock datasets that can be replaced with live APIs later

## Run locally

Open `index.html` in a browser.
